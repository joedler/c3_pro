# GymOS v3.0 健身房管理員操作手冊

本手冊為 GymOS v3.0 的標準作業流程 (SOP) 指南。本手冊將隨著我們測試與確認功能的進度，逐步完善。

---

## 1. 系統權限與職員管理 (Staff Management)

要將一位新進人員設定為「教練」或「管理員」，老闆/主管理員必須直接在 Google 試算表的 `Staff` 分頁進行設定。

### 步驟說明：
1. **獲取真實 LINE ID**：請新進人員先在 LINE 官方帳號內點擊「學員綁定」，當作一般學員隨便輸入姓名綁定。這一步是為了讓系統自動抓取他的真實 `line_uid`。
2. **複製 ID**：打開 `Members` (學員表)，找到他剛才綁定的紀錄，複製他的 `line_uid`。
3. **建立職員檔案**：切換到 `Staff` (職員表)，新增一行並填入以下欄位：

### 欄位填寫規範 (`Staff` 表)：
*   **`staff_id` (職員編號)**：這是內部管理的代號，**由你手動自由編碼**（例如教練可編為 `C001`、`C002`，管理員可編為 `A001`）。
*   **`line_uid` (LINE 唯一識別碼)**：貼上剛才複製的那串真實亂碼 ID。
*   **`real_name` (真實姓名)**：該教練/管理員的顯示名稱。
*   **`role` (系統角色)**：
    *   填入 `admin`：擁有最高權限，可進入管理端「新增班級排程」。
    *   填入 `coach`：僅能進入教練端，進行「出勤校正點名」。
*   **`status` (在職狀態)**：填入 `active` 代表啟用；若員工離職，請改為 `inactive`，系統會立刻撤銷他的所有選單與權限。

當你設定完成後，該人員只要在 LINE 裡面隨便點擊一個選單，系統後台就會瞬間偵測到他已經升級，並自動觸發 LINE API 替換他專屬的圖文選單！

## 4. 一鍵更新 LINE 圖文選單背景圖片 (Rich Menu Synchronization)

為了解決獨立型專案 (Standalone) 無法直接在 Google 試算表選單顯示的問題，我們提供了一套全自動且極度優雅的 **「行動化 / 網頁化一鍵同步」** 機制。你可以在手機或電腦的「管理員控制台」中，隨時更新 LINE 選單圖！

### 第一步：準備精美選單圖片
*   **圖片規格**：請美編製作符合 LINE 官方規定的圖片，尺寸必須剛好是 **`2500 x 843 像素`**，格式為 JPG、JPEG 或 PNG。
*   **上傳至雲端硬碟**：將圖片上傳至你的個人 Google 雲端硬碟，並複製該圖片的 **「共用/分享連結」**（例如：`https://drive.google.com/file/d/xxxxx/view?usp=sharing`）。

### 第二步：貼入試算表 Config 設定
*   打開試算表的 **`Config (系統設定)`** 分頁。
*   你會看到最下方自動新增了三個圖片網址專屬欄位：
    *   **`IMG_MENU_MEMBER`**：學員版選單背景圖網址。
    *   **`IMG_MENU_COACH`**：教練版選單背景圖網址。
    *   **`IMG_MENU_ADMIN`**：管理員版選單背景圖網址。
*   請直接將剛才複製的 Google 雲端硬碟共用連結，貼進對應的 **`value`** 欄位中。

### 第三步：一鍵重新同步起飛 (Web App)
1. 在手機 LINE 或電腦瀏覽器中打開 **`管理控制台 (Admin Mode)`**。
2. 看到最上方的控制台 Hero 區塊中，點擊藍色的 **「🖼️ 同步圖文選單」** 按鈕。
3. 網頁會跳出「⏳ 正在向 LINE 同步覆蓋圖文選單圖...」的提示。
4. 靜候大約 5~10 秒，網頁跳出「🎉 圖文選單重新建立與圖片同步成功！」即可完成！
5. 此時你與所有人員手機 LINE 下方的選單，就已經神不知鬼不覺地完成圖片的視覺更新了！
