# Walkthrough — GymOS v3.0 SaaS 視覺流動無框化與雙軌公告系統

本期更新已完美實現 **SaaS 品牌動態載入**、**去框全寬極大化版面**、**頂部導航簡化與高危選項安全收納**、**首頁摘要 4 大指標卡片與就地展開明細**、**【💰 繳費確認】全面去核點化**，以及 **【📅 課表看板】Google 日曆雙向完美整合與 App 開啟功能**。

---

## 🚀 核心優化項目

### 1. SaaS 品牌與極簡全寬 (Fluid Frameless) 版面
- **動態 SaaS 載入**：前端自動拉取 Google 試算表 `Config` 工作表的品牌標題與 Logo 圖片，自動替換 document title，且支援不同的健身房自訂專屬 Logo 與標題。
- **無邊框版面極大化**：全寬度 `max-w-[98%]` 設計，捨棄了冗餘的 Hero Banner 與厚重的卡片外框，極致釋放螢幕空間，行動裝置貼邊極佳，完全避免表格折行。
- **雙主題一鍵切換**：頂部 Header 整合了 `🌗` 按鈕，點擊即可流暢切換「亮色模式/暗色模式」，並使用極低成本的 CSS 變數 override 完成全站配色轉換，速度極快。
- **設定與高危選項收納**：移除了頂部 Sandbox 角色切換面板，並將「測試沙盒切換」、「資料庫一鍵重置」及「LINE 圖文選單一鍵同步」等高危功能完全收納至右上角「⚙️ 系統設定」彈窗中，避免日常誤觸。

### 2. 【📊 首頁摘要】(Dashboard) 極致進化
- **動態公告跑馬燈 (SaaS 零成本方案)**：首頁頂部新增 HSL 漸層「公告廣播列」，以極致流暢的跑馬燈動態播放最新公告。
- **雙軌公告發布 Modal**：管理員可一鍵開啟 Modal 彈窗發布公告。提供 **100% 免費**的試算表儲存機制；急件時勾選「同步推播至 LINE」，才會調用 API 群發廣播，極大化節省官方推播配額。
- **4 大指標清冊就地展開**：
  - **今日課程數**：點擊卡片可展開今日所有開課明細，包含上課時間、教練、出席人數。
  - **待繳費學員**：點擊卡片直接切換至【💰 繳費確認】分頁。
  - **今日請假數**：點擊卡片可就地展開請假人名、班級與請假原因。
  - **今日補課數**：點擊卡片可就地展開補課學員姓名與預約目標班級。

```mermaid
graph TD
  A[管理員發布公告] --> B{勾選同步 LINE?}
  B -- Yes --> C[儲存至試算表 & LINE API 廣播]
  B -- No --> D[僅儲存至試算表 (100% 免費)]
  C --> E[學員 LINE 實時收到公告]
  D --> F[網頁公告跑馬燈實時同步]
```

### 3. 【💰 繳費確認】(Tuition) 命名去核點化
- 全面更名為 **【💰 繳費確認】**，移除所有可能導致學員誤解的「核點」字樣。
- 保留 Checkbox 批量選取與「🚀 批量確認已繳費」全域按鈕。

### 4. 【📅 課表看板】(Calendar) 雙向完美整合
- 網頁端採用 FullCalendar 與後端 Google 日曆即時連動，呈現極致暗黑/亮色主題風格。
- 整合「📅 在 Google 日曆 App 中開啟」外跳按鈕，實現與行動端日曆 App 的無縫銜接。
