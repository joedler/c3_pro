# 🏋️ GymOS 3.0 會員功能使用指南 (學員手冊)

歡迎加入！本指南將引導您如何在 LINE 官方帳號中，以最直覺的方式使用您的專屬課表、線上請假與跨班補課功能。

---

## 🚀 步驟一：一秒綁定會員帳號 (新手開戶)

當您第一次點選 LINE 選單中的 **「📅 我的課表」** 時，系統會自動引導您進行身分啟動：

```
[ 📅 我的課表 ] (點擊) ➡️ [ 👤 輸入健康基本資料 ] ➡️ [ 🏋️ 選擇預約班型 ] ➡️ [ 🎉 綁定完成 ]
```

1. **基本資料填寫**：輸入您的真實姓名、性別、生日、身高與體重，這將同步為您建立健身房的健康檔案。
2. **選擇方案與時段**：勾選您要報名的課程方案（如 A 類 12週、B 類 1個月）以及您期望的上課時段。
3. **確認啟動**：點擊「確認綁定並預約」，系統便會即時與後台對接。
4. **帳號解鎖**：未來您點擊選單將直接**無縫直達個人主頁**，不再需要重複登入或輸入資料！

---

## 🚫 步驟二：線上請假 (快速請假與保留額度)

當您臨時有事不克前來時，請在**下課前**在 LINE 中一秒送出請假：

1. 點選選單的 **「✍️ 我要請假」**（或進入「我的課表」點擊該堂課程）。
2. 畫面會列出您 **未來 4 週內即將上課的所有時段**。
3. 勾選您欲請假的該堂課程，並在下方寫下原因（如：加班、身體不適，此項選填）。
4. 點擊 **「送出請假申請」**。
5. 系統會立刻為您**保留 1 點補課點數**，並實時從該堂課的 Google 日曆名單中移除您，完全自動化！

> 💡 **溫馨提示**：
> * 請假次數無上限，只要在下課前送出，皆可 100% 保留補課額度。

---

## 🔄 步驟三：跨班補課預約 (點數實時兌換)

當您想要使用請假保留的額度進行補課時，操作極為方便：

1. 點選選單的 **「⚡ 我要補課」**。
2. **第一步：選擇額度**
   * 列表會顯示您所有「已請假、且尚未補課」的保留額度庫。勾選您想使用的那一張點數。
3. **第二步：選擇補課時段**
   * 系統會根據您的運動程度（如 L1），**實時篩選出同等難度、且目前仍有名額的未來時段**！
   * 每張卡片右側會高亮顯示 `剩餘 X 位` 名額。勾選您最心儀的時段。
4. **確認預約**：點擊下方 **「確認跨班補課預約 (不可取消)」**，即預約成功！
   * 系統會即時將您加入目標課堂的 Google 日曆出席名單中，鎖定您的專屬名額！

---

## ❓ 常見問題 FAQ

### Q1：我請假之後，補課點數有期限嗎？
* **A1**：您的補課額度期限與您報名的方案類型一致。B 類（1個月方案）學員需在當月使用完畢；A 類與 C 類（12週方案）學員則可於整期課程結束前彈性使用。

### Q2：補課預約成功後，可以取消或更改嗎？
* **A2**：為了保障所有學員的公平搶課權利與名額防爆管控，**補課預約一旦送出，便無法自行取消或更改**。請在送出前仔細核對您的時間喔！

### Q3：為什麼我點擊「我要補課」，第二步沒有顯示任何時段？
* **A3**：這代表目前其他同難度的班級在未來時段均已「額滿」，或是暫時沒有開課。名額都是即時動態更新的，建議您隔日再上來查看是否有其他同學請假釋出的名額喔！
