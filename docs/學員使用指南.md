# 🏋️ GymOS 3.0 會員功能使用指南 (學員手冊)

歡迎加入 C3 Fitness！本指南將引導您如何在 LINE 官方帳號中，以最直覺的方式使用您的專屬課表、線上請假與跨班補課功能。

---

## 🚀 步驟一：一秒綁定會員帳號 (新手開戶)

當您第一次點選 LINE 選單中的 **「📅 我的課表」** 時，系統會自動引導您進行身分啟動：

```
[ 📅 我的課表 ] (點擊) ➡️ [ 👤 輸入健康基本資料 ] ➡️ [ 🏋️ 選擇預約班型 ] ➡️ [ 🎉 綁定完成 ]
```

1. **基本資料填寫**：輸入您的真實姓名、性別、生日、身高與體重，這將同步為您建立健身房的健康檔案。
2. **選擇方案與時段**：從課程清單中選擇您要報名的時段。
3. **確認啟動**：點擊「確認綁定並預約」，系統便會即時與後台對接。
4. **帳號解鎖**：未來您點擊選單將直接**無縫直達個人主頁**，不再需要重複登入或輸入資料！

---

## 📊 步驟二：了解您的課程統計數據

進入「我的課表」後，首頁會顯示您目前這期課程的出勤儀表板：

```
┌─────────────────────────────┐
│ 👤 王小明 的課程資訊          │
├─────────────────────────────┤
│ 📅 班級：混合重訓 週一三晚A班 │
│ ✅ 已上：6 堂               │
│ ❌ 請假：2 堂               │
│ 🔄 補課：1 堂               │
│ 💪 可補額度：1 堂            │
│ 📊 剩餘堂數：N 堂            │
└─────────────────────────────┘
```

### 各數字計算方式

| 數字 | 計算公式 | 說明 |
|------|---------|------|
| **已上堂數** | 目前為止已經過的課堂數 | 不含請假 |
| **請假堂數** | 您申請且通過的請假筆數 | |
| **可補額度** | 請假堂數 − 已登記補課次數 | 這才是您「還能預約補課」的點數 |
| **剩餘堂數** | 已繳費總堂數 − 已上堂數 | 這期還剩幾堂課 |

### 關於 B 類 (混合重訓) 課程的堂數說明

B 類課程以**月份**為計費單位，**每月實際堂數依照月曆動態計算**，因此不同月份的總堂數可能不同：
- 例：2026 年 5 月（週一+週三），共出現 9 個上課日 → 本月 **9 堂**
- 例：2026 年 6 月（週一+週三），可能共出現 8 個上課日 → 下月 **8 堂**

> 💡 系統會自動在每月開始時計算正確堂數，您只需繳費後等待管理員確認即可。

---

## 🚫 步驟三：線上請假 (快速請假與保留額度)

當您臨時有事不克前來時，請在**下課前**在 LINE 中一秒送出請假：

1. 點選選單的 **「✍️ 我要請假」**（或進入「我的課表」點擊該堂課程）。
2. 畫面會列出您 **未來 4 週內即將上課的所有時段**。
3. 勾選您欲請假的該堂課程，並在下方寫下原因（如：加班、身體不適，此項選填）。
4. 點擊 **「送出請假申請」**。
5. 系統會立刻為您**保留 1 點補課點數**，並實時從該堂課的 Google 日曆名單中移除您，完全自動化！

> 💡 **溫馨提示**：
> * 請假次數無上限，只要在下課前送出，皆可 100% 保留補課額度。
> * **B 類（月費制）學員**：補課點數在當月有效，下月換期後不延續，請把握時間安排補課！

---

## 🔄 步驟四：跨班補課預約 (點數實時兌換)

當您想要使用請假保留的額度進行補課時，操作極為方便：

1. 點選選單的 **「⚡ 我要補課」**。
2. **第一步：選擇額度**
   * 列表會顯示您所有「已請假、且尚未補課」的保留額度庫。勾選您想使用的那一張點數。
3. **第二步：選擇補課時段**
   * 系統會根據您的運動程度（如 Lv.4），**實時篩選出同等難度、且目前仍有名額的未來時段**！
   * 每張卡片右側會高亮顯示 `剩餘 X 位` 名額。勾選您最心儀的時段。
4. **確認預約**：點擊下方 **「確認跨班補課預約 (不可取消)」**，即預約成功！
   * 系統會即時將您加入目標課堂的 Google 日曆出席名單中，鎖定您的專屬名額！

## ❓ 常見問題 FAQ

### Q1：我的「可補額度」為什麼是 0？
* **A1**：「可補額度」的計算公式是：**請假堂數 − 已登記補課次數**。如果您已對所有請假登記了補課，額度即為 0。另外，若您的請假申請尚在審核中（非 `已批准` 狀態），也不會計入可補額度。

### Q2：請假之後，補課點數有期限嗎？
* **A2**：
  - **A 類、C 類（12 週固定方案）**：補課額度在整期課程（12 週）結束前均有效，可彈性安排。
  - **B 類（月費制方案）**：補課點數在**當月有效**，換期後不延續至下月，請盡快安排補課。

### Q3：補課預約成功後，可以取消或更改嗎？
* **A3**：為了保障所有學員的公平搶課權利與名額防爆管控，**補課預約一旦送出，便無法自行取消或更改**。請在送出前仔細核對您的時間喔！

### Q4：為什麼我點擊「我要補課」，第二步沒有顯示任何時段？
* **A4**：這代表目前其他同難度的班級在未來時段均已「額滿」，或是暫時沒有開課。名額都是即時動態更新的，建議您隔日再上來查看是否有其他同學請假釋出的名額喔！

### Q5：我的課表資訊顯示的數據感覺不對，怎麼辦？
* **A5**：如果您剛完成請假或補課登記，建議先退出「我的課表」後**重新點選進入**（清除 LINE App 內部快取），若數據仍不正確請聯絡管理員協助確認。

### Q6：為什麼我看不到右上角齒輪設定？
* **A6**：齒輪設定是管理端專用功能，學員端不需要設定 API、推播開關或版本維護工具，因此正式環境不會顯示。學員只需要透過 LINE 圖文選單進入「我的課表」、「我要請假」、「我要補課」或「加選課程」即可。

### Q7：我需要自己儲存什麼系統設定嗎？
* **A7**：不需要。學員端的資料會由系統依照您的 LINE 身分自動讀取與更新；若畫面資料看起來沒有更新，請先重新開啟 LINE 官方帳號視窗，仍有問題再聯絡管理員協助。
